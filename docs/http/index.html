Kapitel 4 - HTTP

HTTP steht für Hyper Text Transfer Protocol und wird verwendet für die Übertragung von HTML, der Hyper Text Markup Language.

Das HTTP-Protokoll wurde einfach gehalten und hat sich inzwischen als bewährtes Mittel für die Kommunikation allgemein herausgestellt, nicht für den Austausch von HTML-Dokumenten. Beispielsweise gibt es HLS, was für HTTP Live Streaming steht und wobei Video-Streams per HTTP übertragen werden.

HTTP ist standardisiert per RFC 9110.

In diesem Kapitel wollen wir uns HTTP etwas genauer anschauen und verstehen.

info

Die Browser-Dev-Tools verfügen auch über einen Netzwerk-Tab. Dort können wir einsehen, welche Requests mit welchen Daten gesendet wurden und welche Antwort darauf zurück kam.
