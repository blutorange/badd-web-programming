JSON-Formatierer

Zur Unterstützung bei der Entwicklung soll ein kleines Tools entwickelt werden, welches die Eingabe eines JSON erlaubt und dieses formatiert anzeigt.

Beschreibung​

  • 1 Textarea für die Eingabe des JSON
  • 1 Eingabefeld für die Anzahl der Leerzeichen pro Einrückungslevel
  • 1 Button zum Formatieren

Mindestanforderungen​

  • Das formatierte JSON soll direkt in der Textarea angezeigt werden.
  • Bei ungültigen JSON soll eine Meldung angezeigt werden.
  • Für das Einrückungslevel sollen Werte zwischen 1 und 10 erlaubt sein. Initial soll der Wert 4 im Feld stehen.

Hinweise​

Das globale Objekt JSON stellt bereits entsprechende Funktionen für die Arbeit mit JSON bereit.
