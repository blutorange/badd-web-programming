Dynamische Formularerstellung

Anhand von JSON-Daten soll ein Formular erstellt und entsprechend mit Interaktion versehen werden.

Mindestanforderungen​

  • 1 Eingabefeld für eine JSON-Datenstruktur, welche das Formular darstellt.
  • 1 Button Formular erzeugen, der anhand der JSON-Daten ein interaktives Formular erzeugt.
  • Die konkrete Struktur des JSON kann beliebig festgelegt werden. Unten ist ein mögliches Beispiel.
  • Es sind wenigstens alle Features zu unterstützen, welche im Beispiel unten verwendet werden.

Obiges JSON würde etwa folgendes Formular erzeugen:

Konkret muss das Formular aus mehreren Fieldsets bestehen können, wobei jedes Fieldset aus Zeilen besteht. Jede Zeile enthält ein oder mehrere Formularfelder. Enthält eine Zeile mehrere Formularfelder, werden diese nebeneinander angeordnet.

Jedes Formularfeld hat eine eindeutig ID, sowie ein Label und ist entweder ein Pflichtfeld oder nicht. Folgende Arten von Formularfeldern sollen unterstützt werden.

  • Einzeiliges Eingabefeld, mit Festlegung des Datentyps.
  • Mehrzeiliges Eingabefeld
  • Checkbox, die angehakt oder abgehakt ist
  • Einfache Auswahl, die nur die Auswahl 1 Option erlaubt
  • Mehrfache Auswahl, die die Auswahl mehrere Optionen erlaubt
  • Berechnungsfeld, dessen Wert sich anhand einer vom Nutzer eingegebenen Formel ergibt. Das der Wert berechnet ist, darf der Nutzer den Wert nicht manuell ändern. Die Formel ist eine JavaScript-Expression, welche in einem Scope ausgewertet wird, wo die Symbole numberValue und stringValue zur Verfügung stehen. Der Wert ist jeweils eine Funktion, welche die ID eines Formularfelds entgegennimmt und den Wert von diesem Feld zurückliefert, entweder als Zahl oder als String.

Hinweise​

Zum Ausführen von JavaScript kann ein <script>-Element erzeugt werden, welches in den <head> eingehangen wird und direkt danach wieder entfernt wird. Um den Wert der Expression abzugreifen, kann dieser temporär in eine globale Variable geschrieben werden. Um Symbole im Scope der Expression verfügbar zu machen, kann die Expression in einer IIFE gewrapped werden, welche die Symbole als Funktionsparameter deklariert.

Vor Beginn dieser Aufgabe sollte man sich die Datenstruktur gut durchdenken und damit vertraut machen.
