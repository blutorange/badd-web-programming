Aufgaben zu HTML

Aufgabe 1 - Hallo Welt​

Erstellen Sie eine "Hallo Welt"-Seite! Die Seite soll:

  • Den doctype deklarieren
  • Die Dokumenten/Sprache auf Deutsch setzen
  • Zeichensatz-Encoding auf UTF-8 setzen
  • Den Titel "Hallo Welt" haben
  • Den Text "Hallo Welt" anzeigen

Diese Seite kann Ihnen für andere Aufgaben als Vorlage dienen.

Aufgabe 2 - Überschriften​

Erstellen Sie eine HTML-Seite mit Überschriften verschiedener Ebene! Unter jeder Überschrift sollen mehrere Absätze Texte stehen.

Sie können auch den Lorem Ipsum-Generator zum Erzeugen von Text verwenden.

Aufgabe 3 - Links​

Erstellen Sie eine HTML-Seite mit Links auf:

  • externe Seiten, die sich im selben Tab öffnen
  • externe Seiten, die sich in einem anderen Tab öffnen
  • eine andere von Ihnen erstellte Seite
  • einen Abschnitt weiter unten auf der gleichen Seite

Die offizielle Demo-Seite https://example.com kann immer verlinkt werden, ohne nach Erlaubnis zu fragen.

Aufgabe 4 - Listen​

Erstellen Sie eine HTML-Seite mit verschiedenen Arten von Listen und füllen Sie diese mit passendem Beispielsinhalt!

  • geordnete Liste
  • ungeordnete Liste
  • Definitionsliste

In anderen Worten: Überlegen Sie sich, für welche Art von Inhalt diese verschiedenen Listen verwendet werden können.

Aufgabe 4 - Bilder​

Erstellen Sie eine HTML-Seite mit verschiedenen Bildern! Beim Klicken auf ein Bild soll sich eine von Ihnen gewählte Webseite öffnen. Stellen Sie sicher, dass die Bilder auch einen alternativen Beschreibungstext enthalten!

Bei Wikimedia Commons finden Sie Bilder, Audio und Video, die sie als Beispiel verwenden können (Lizenzbedingungen beachten!).

Aufgabe 5 - Audio​

Erstellen Sie eine HTML-Seite, wo Sie eine Audio-Datei einbinden! Die Audio-Datei soll beim Laden der Seite nicht automatisch abgespielt werden, sondern erst, nachdem der Nutzer auf den Play-Button gedrückt hat.

Bei Wikimedia Commons finden Sie Bilder, Audio und Video, die sie als Beispiel verwenden können (Lizenzbedingungen beachten!).

Aufgabe 6 - Video​

Erstellen Sie eine HTML-Seite, wo Sie ein Video einbinden! Das Video soll beim Laden der Seite nicht automatisch abgespielt werden, sondern erst, nachdem der Nutzer auf den Play-Button gedrückt hat. Bis zum Abspielen soll initial ein von Ihnen gewähltes Titelbild angezeigt werden.

Bei Wikimedia Commons finden Sie Bilder, Audio und Video, die sie als Beispiel verwenden können (Lizenzbedingungen beachten!).

http://localhost:3000/badd-web-programming/docs/html/body/#semantisches-html

Aufgabe 7 - Semantische HTML-Elemente​

Erstellen Sie eine HTML-Seite mit verschiedenen semantischen HTML-Elementen! Überlegen Sie sich entsprechenden Inhalt, der zur Bedeutung dieser HTML-Element passt.

Im Kapitel zu Semantisches HTML können Sie noch einmal nachschauen, welche Elemente es gab.

Aufgabe 8 - Tabelle​

Setzen Sie die Table 3 aus dem Paper Evaluating the Generalization Capabilities of Large Language Models on Code Reasoning mittels HTML um!

(Quelle: https://arxiv.org/abs/2504.05518)

Aufgabe 9 - Kontaktformular​

Erstellen Sie eine HTML-Seite mit einem Kontaktformular! Das Formular sollte wenigstens ein Fieldset sowie die Eingabe von Name, E-Mail und Anliegen enthalten. Zudem muss ein Button zum Absenden enthalten sein.

Sie können als Form-Action auch Postman Echo verwenden, um die abgesendeten Daten einzusehen:

https://postman-echo.com/post

Aufgabe 10 - Auswahlliste​

Erstellen Sie eine HTML-Seite mit einem Auswahlfeld für eine Programmiersprache. Der Nutzer soll eine der möglichen Optionen auswählen können.

Eine Liste von Programmiersprachen finden Sie etwa im Wikipedia-Artikel List of programming languages by type.

Versuchen Sie auch, die Auswahloptionen nach dem Typ der Programmiersprache zu gruppieren!

Aufgabe 11 - Abbildung​

Erweitern Sie Ihre Lösung von 4, indem Sie das Bild oder die Bilder als Abbildung mit Bildunterschrift und Quellenangabe umwandeln!
